# 数字电路仿真平台 (Digital Circuit Simulation)

这是一个基于 Next.js 的数字电路仿真网站，专为《数字与逻辑电路课程 DSR》设计，用于直观展示经典 74LS 系列集成电路的功能与逻辑。

## 🌟 功能特点

- **经典芯片库**:
  - **74LS148**: 8线-3线优先编码器
  - **74LS138**: 3线-8线译码器
  - **74LS151**: 8选1数据选择器
  - **74LS283**: 4位二进制全加器
  - **74LS90**: 二-五-十进制计数器 (支持 **8421/5421** 码配置)
  - **74LS163**: 4位二进制同步计数器
  - **74LS194**: 4位双向移位寄存器 (支持 **循环左移/右移**)

- **交互式仿真**:
  - 实时点击开关控制输入电平 (高/低)。
  - 动态 LED 显示输出状态。
  - 支持手动触发 CP 时钟脉冲。
  - **高级配置**: 针对特定芯片（如 74LS90, 74LS194）提供额外的连线配置开关，模拟实际电路中的反馈连线。

- **界面设计**:
  - 清晰的白底黑字风格，适合教学演示。
  - 粗线条边框与大字体，增强可视性。
  - 实时访客计数器（右下角）。

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
访问地址: **http://localhost:16390**

### 3. 构建与生产运行
```bash
npm run build
npm start
```
生产环境同样运行在 **16390** 端口。

## 🛠️ 技术栈

- **框架**: [Next.js 14](https://nextjs.org/) (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图形**: SVG (用于绘制芯片引脚与逻辑图)

## 📂 项目结构

- `app/`: 页面路由与布局
  - `page.tsx`: 主页面逻辑
  - `layout.tsx`: 全局布局与访客计数器集成
- `components/`: React 组件
  - `ChipSimulator.tsx`: 核心仿真器组件，负责渲染芯片与处理交互
  - `VisitorCounter.tsx`: 访客计数器组件
- `lib/`: 核心逻辑
  - `chip-definitions.ts`: 所有芯片的引脚定义、逻辑真值表与状态机实现
  - `visitor-store.ts`: 访客数据存储逻辑
- `public/`: 静态资源 (背景网格等)

## 📝 特殊功能说明

- **74LS90**: 底部设有 `QA->CP1` (8421码) 和 `QD->CP0` (5421码) 开关，用于演示不同进制计数。
- **74LS194**: 顶部设有 `Q3->DSR` 和 `Q0->DSL` 开关，开启后可演示循环移位效果。
- **访客统计**: 包含防刷机制，同一设备 10 分钟内仅记录一次访问。
